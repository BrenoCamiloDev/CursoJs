## Como Navegar entre os elementos e selecionar:

1. #### Marca:
```
getElementsByTagName()
```
2. #### ID:
```
getElementById()
```
3. #### Nome:
```
getElementsByName()
```
4. #### Classe:
```
getElementsByClassName()
```
5. #### Seletor:
```
querySelector()
querySelectorAll()
```
